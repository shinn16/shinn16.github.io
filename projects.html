Projects

I have a variety of projects going on at the moment, most of which you can actively track on GitLab.

  1. IP-Tracker

    The purpose of the software is to check the current IP address of the network it is on and record it. In the event that the address changes, the program will send an email out to inform the server owner of the change. If NextCloud is run on the server, the config file will automatically be updated to allow access from the new address. This software is also capable of tracking local IP addresses, but this has been specialized for Raspberry Pis running Raspian. In order to run this software, the system will need at least python 3.4 installed. You will also need to install git on the system to ensure that the scripts run properly. IP-Tracker was designed in a way that if you have python 3, pip3, and git installed, you should need no additional packages for it to run properly.

  1. Firmware Forge

    Firmware Forge is a Flask based web application that allows users to manage and remotely upload firmware to ESP8266 based devices. This was initially built as a pet project of mine, but I have decided to make it open to the public to use. It is still really new and has a lot of bugs to work out, but with some time and testing this will hopefully mature into a useful tool for IoT enthusiasts and the ESP8266 community. There is an accompanying library I have written to handle the updating process. It too is still in its infancy, but will soon be available on platformio.

  2. NodeMCU Climate Sensor

    This is a simple sensor built on the ESP8266 platform. It is equipt with a DHT22 digital humidity and temperature sensor. Using MQTT, the sensor sends climate data to a server. From there you can use something like Home Assistant to automate parts of your home using climate data. I use mine as a cheap thermostat in my apartment. The code is freely available, but was written to suite my needs. I did put some effort into making this a plug and play solution for most cases, but any specialized implementations are entirely up to you if you plan on using the code as a base. Enjoy!

In order for them to work, they will need a few edits on a system by system base. I built them to work on my computer, so they are specialized for my particular needs.
